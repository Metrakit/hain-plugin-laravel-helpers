A plugin for find fastly a Laravel helper in the [Hain](https://github.com/appetizermonster/hain) app (Alfred clone for Windows).

## Install

Type this command in the Hain input :
```
/hpm install laravel-helpers
```

## Usage

```
/l5 a Laravel helper
```
